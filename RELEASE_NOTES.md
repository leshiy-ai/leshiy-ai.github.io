# Leshiy-AI Ecosystem: Релиз v1.1.2 от 07.04.2026 — «Grok и реставрация фото»

Этот релиз приносит два крупных обновления: долгожданный Image-to-Image для Pollinations, который теперь умеет реставрировать и стилизовать фото, и интеграцию новой мощной модели Grok.

### 🎨 Главное: Полноценный Image-to-Image (I2I) для Pollinations

Мы наконец-то «дожали» Pollinations. Теперь это не просто генерация картинок по тексту, а мощный инструмент для работы с вашими файлами.

*   **Модель Pruna (p-image-edit):** Внедрена поддержка специализированной модели для редактирования. Она реально видит исходное изображение.
*   **Режим реставрации:** Теперь «Леший» умеет восстанавливать старые фото, раскрашивать ч/б снимки и убирать дефекты, сохраняя лица и композицию.
*   **Стилизация:** Изменение цвета объектов, перерисовка фона и корректировка деталей (теперь «черная шерсть» — это реально черная шерсть).
*   **Оптимизация передачи:** Перешли на строгий `FormData` (Multipart) для эндпоинта `/edits`, что обеспечило стабильную передачу бинарных данных.

### 🧠 Интеллект и Чат

*   **Интеграция Grok (xAI):** В список доступных моделей добавлен `Grok-4-1-fast`. Умный, дерзкий и чертовски быстрый.

### 🛠️ Технические правки и фиксы

*   **Исправлен BotHub I2I:** Решена проблема с ошибкой `502 Bad Gateway` при загрузке больших изображений. Запросы теперь идут напрямую, минуя внутренние прокси.
*   **Отказоустойчивость прокси:** Улучшена логика работы с прокси в Yandex Cloud. При возникновении таймаутов (`504 Gateway Timeout`) система автоматически пробует резервные пути, гарантируя получение ответа.
*   **Fixed:** Исправлена критическая ошибка `ReferenceError: response is not defined`, возникавшая при генерации изображений.
*   **Fixed:** Устранен баг с игнорированием вложений (теперь Pollinations не «галлюцинирует» с нуля, а опирается на прикрепленный файл).
*   **Fixed:** Восстановлен текстовый чат с Pollinations после исправления ошибки с неверным URL.
*   **Economy:** Оптимизированы расходы — стоимость одной реставрации через Pollinations теперь составляет всего 0.01 pollen (~$0.0001).
