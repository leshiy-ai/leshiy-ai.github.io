# Leshiy-AI Ecosystem: Релиз v1.1.4 — «Стабильный Android APK»

Этот релиз сфокусирован на улучшении доступности и стабильности приложения на мобильных устройствах.

### 🚀 Главное: Стабильный APK для Android

Мы рады представить официальный и стабильный APK-файл для Android, созданный с помощью Bubblewrap. Это делает установку и использование «Leshiy-AI» на Android-устройствах проще и надежнее.

*   **Простая установка:** Больше не нужно полагаться на веб-версию, теперь есть полноценное приложение.
*   **Оптимизация:** Приложение оптимизировано для мобильных устройств, что обеспечивает более плавную работу.
*   **Надежность:** Сборка через Bubblewrap гарантирует соответствие последним стандартам Android и лучшую интеграцию с системой.
