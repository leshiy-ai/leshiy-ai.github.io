# Leshiy-AI Ecosystem: Релиз v1.1.0 от 01.04.2026 — «Бесшовный Мир и Умный UX»

Этот релиз превращает наш сайт в полноценную мультиплатформенную экосистему. Теперь твой ИИ-помощник не просто вкладка в браузере, а единый организм, живущий в Telegram, VK и Web одновременно.

### 🧠 Сквозная история и S3-контекст: Твой ИИ больше не забывает тебя и историю твоих сообщений.

Мы ушли от локального хранения данных. Теперь вся переписка и состояние сессии живут в Yandex Object Storage (S3).

*   **Экосистема Telegram (Bot + TWA + Web):** Начал диалог с ботом в обычном чате? Открой Web App — история уже там. Перешел на сайт? Контекст подтянулся мгновенно.
*   **Экосистема VK (Bot + Mini App + Web):** Бесшовный переход между чатом ВК и мини-приложением. Общий S3-бакет гарантирует доступность твоих данных 24/7.
*   **Smart Identity:** Сайт теперь — это полноценный хаб. Система идентификации связывает твои аккаунты в соцсетях, позволяя Gemini оперировать всей глубиной истории запросов.

### 🎮 Новый UX: Управление в один клик

Интерфейс стал отзывчивее и умнее. Мы переосмыслили навигацию:

*   **Интерактивное меню режимов:** Быстрое переключение между задачами (Чат, Фото, Аудио, Видео).
    *   **Короткое нажатие:** Циклическая смена режима.
    *   **Долгое нажатие:** Открывает подменю для прямого выбора.
*   **Умный Сайдбар:** Боковая панель теперь уважает твое пространство и автоматически сворачивается при клике в пустую область.
*   **Микрофон 2.0:** Обновленный интерфейс записи для более комфортной работы с голосом.

### ✨ Интеллектуальный Менеджмент Чат-сессий

Больше никаких «Новых чатов» в списке.

*   **Авто-именование:** Система сама проанализирует контекст и даст название диалогу после 3-го сообщения.
*   **Кнопка Магии (✨):** Хочешь переименовать чат прямо сейчас? Одна кнопка — и Gemini подберет идеальное название на основе текущей темы.

### 🛠 Технические правки

*   Глубокая оптимизация `App.jsx`, `App.css` и `index.html`.
*   Улучшена работа с бинарными данными и медиа-контентом.
*   Исправлены задержки при старте приложения (добавлен асинхронный импорт и `defer` для внешних SDK).
