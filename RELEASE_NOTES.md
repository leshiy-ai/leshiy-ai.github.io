## версия 1.0.9

Это крупное обновление, сфокусированное на улучшении пользовательского опыта (UX) и добавлении новых способов взаимодействия с AI.

### Что нового ✨

- **🚀 Продвинутое Меню режимов (Modes Menu):**
  - **Циклическое переключение:** Краткое нажатие на иконку режима (например, 1️⃣) циклически переключает доступные функции (Общение -> Генерация фото -> Работа с аудио и т.д.).
  - **Подменю выбора:** Долгое нажатие (long-tap) на иконку режима открывает подменю для быстрого и прямого выбора нужной функции. Меню остается открытым, но циклическое переключение продолжает работать, подсвечивая активный пункт.

- **🎤 Улучшенная работа с микрофоном:**
  - **Запись в файл (Voice Memo):** Долгое нажатие на иконку микрофона теперь активирует режим записи голосового сообщения. После окончания записи, аудиофайл прикрепляется к сообщению.
  - **Авто-переключение в режим транскрибации:** После записи в файл, приложение автоматически переключается в режим "3️⃣ - Работа с аудио", чтобы вы могли сразу продолжить отправлять голосовые на распознавание.
  - **Упрощенная запись:** В режиме "3️⃣" для начала записи достаточно одного короткого клика по микрофону.

- **🖼️ Интерактивные превью для медиа:**
  - **Просмотр фото:** Для прикрепленных картинок и фото теперь есть возможность предпросмотра в новом окне.
  - **Прослушивание аудио:** Перед отправкой записанное голосовое сообщение можно прослушать прямо в панели превью.
  - **Просмотр видео:** Для прикрепленных видеофайлов теперь отображается небольшой проигрыватель, позволяющий посмотреть содержимое прямо в превью.

### Улучшения 🎨

- **Умный сайдбар:** Боковая панель (Sidebar) теперь автоматически сворачивается, если кликнуть в любом месте за ее пределами, что делает интерфейс чище и удобнее.
- **Обновление интерфейса:** Значительные изменения в `App.jsx`, `App.css` и `index.html` для поддержки нового меню, превью и улучшенного микрофона.
- **Оптимизация логики:** В `leshiy-core.js` добавлена новая логика для обработки сложных взаимодействий с меню и микрофоном.
