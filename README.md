# 🌲 Leshiy-AI: Единая Экосистема Вашего ИИ

Leshiy-AI превратился из простого ассистента в полноценную мультиплатформенную экосистему. Ваш ИИ-помощник теперь — это единый организм, живущий в **Telegram, VK и Web одновременно**. 
Приложение объединяет мощь нейросетей (Gemini, Cloudflare AI) и удобство управления файлами через социальные экосистемы (VK Mini Apps и TG App).

### 🧠 Сквозная история и S3-контекст
Забудьте о разрозненных диалогах. Вся история переписки и состояние сессии теперь хранятся в облаке (Yandex Object Storage), обеспечивая **бесшовный опыт**:
- **Единый контекст:** Начните диалог с ботом в вк-чате, продолжите в VK Mini App и завершите на сайте — ИИ будет помнить всё. Тоже самое с Telegram.
- **Smart Identity:** Система идентификации связывает ваши аккаунты по ID, позволяя Gemini оперировать всей глубиной истории запросов для более точных и персонализированных ответов.

## ✨ Основные возможности

### 🤖 Интеллектуальный ассистент

- Мультимодальность: Распознавание текста, изображений, аудио и видео (Gemini 2.5 Pro/Flash).
- **Меню режимов:** Быстро переключайтесь между задачами (Общение, Генерация фото, Работа с аудио, Видео) с помощью нового интерактивного меню. Доступно как циклическое переключение, так и прямое через подменю по долгому нажатию.
- **Микрофон 2.0:** При обычном клике микрофона включается режим распознавания голоса в текст и можно просто надиктовывать сообщение. По долгому нажатию записывайте голосовые заметки. Файл прикрепится автоматически, и режим переключится на "3️⃣ - Работу с аудио".
 **Умный Сайдбар:** Боковая панель теперь уважает ваше пространство и автоматически сворачивается при клике в пустую область.
- **Интерактивные превью:** Прослушивайте аудио и просматривайте видео прямо в панели превью перед отправкой.
- **Мультимодальность:** Распознавание текста, изображений, аудио и видео (Gemini 2.5 Pro/Flash).
- **Универсальное распознавание:** Отправляйте изображения, аудио- и видеофайлы даже без сопроводительного текста. AI проанализирует медиафайл и даст ответ (например, опишет картинку или транскрибирует речь).
- **Умный контекст:** Поддержка системных промптов и ведение диалога в дружелюбном стиле.
- **Автоматическое именование чатов**: Диалоги автоматически получают осмысленное название после 3 сообщений. Добавлена кнопка (✨) для принудительного переименования в любой момент.
- Динамическое меню: Интерактивные кнопки для управления функциями прямо в чате.

### 🗄 Хранилка (Storage Gateway)

- **Сохранение по ключевому слову:** Просто прикрепите файлы к сообщению и напишите "сохрани", чтобы мгновенно загрузить их в ваше облачное хранилище.
- Мост в облака: Прямая загрузка файлов на Яндекс Диск, Google Drive, Dropbox, Mail.ru (WebDAV) и частные FTP/SFTP сервера - прямое взаимодействие с **"Хранилкой"**
- Работа с архивами: Поддержка .zip, .rar, .7z, документов и медиафайлов.
- Реферальная система: Возможность поделиться доступом к своей папке в облаке через инвайт-ссылки.
- Zero-Storage Upload: Файлы не хранятся на промежуточных серверах, а пробрасываются в Ваше облако через бинарные потоки (ArrayBuffer и Stream).

## 🚀 Технический стек

### Frontend (React)

- ReactMarkdown: Для красивого отображения ответов нейросети.
- VK ID SDK и Telegram ID: Бесшовная авторизация пользователей.
- **Улучшенный интерфейс**: В панель чатов добавлены кнопки для быстрого ручного (✏️), автоматического (✨) и удаления (🗑️) чата.
- Swipe-to-Delete: Жестовое управление сообщениями.
- Multi-file Preview: Визуальное отображение иконок типов файлов (🎬, 🎵, 📦, 📄) перед отправкой.

### Backend (Yandex Cloud / Gateway)

- Proxy Server: Безопасное хранение API-ключей и обход CORS.
- KV Storage: Хранение токенов авторизации и пользовательских настроек (выбранная папка, квоты).
- S3 Backet: Хранение истории сообщений для поддержки мультиплатформенного контекста.
- Binary Stream Handler: Эффективная обработка больших файлов без перегрузки памяти.

## 🛠 Установка и настройка

1.  **Клонируйте репозиторий:**
    ```bash
    git clone https://github.com/your-repo/leshiy-ai.git
    ```

2.  **Настройте конфиг (`config.js`):**
    Укажите URL вашего прокси-сервера и секретные ключи:
    ```javascript
    export const CONFIG = {
        PROXY_URL: 'https://your-proxy.workers.dev',
        STORAGE_GATEWAY: 'https://your-storage.workers.dev',
        ADMIN_CHAT_ID: 'ваша_id'
    };
    ```

3.  **Сборка проекта:**
    ```bash
    npm install
    npm run build
    ```

## 🔒 Безопасность

- Приложение использует VK ID для идентификации, что исключает необходимость хранения паролей.
- Приложение использует Telegram для идентификации, для совместимости с Телеграм-ботом.
- Все запросы к AI проходят через защищенный прокси с проверкой X-Proxy-Secret.
- Прямая передача байтов (Buffer) обеспечивает максимальную приватность ваших данных при загрузке.
